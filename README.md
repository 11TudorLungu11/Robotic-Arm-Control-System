# Robotic Arm Control System

## Overview

This project presents an Arduino-based robotic arm designed for object manipulation using multiple servo motors.

The robotic arm was developed as a university project to better understand robotic mechanisms, PWM control, servo motor programming and hardware-software integration.

---

## Hardware Components

- Arduino Uno
- Servo motors
- Robotic arm kit
- Breadboard
- Jumper wires
- Push buttons
- LEDs
- External power supply

---

## Main Features

- Multi-axis robotic arm movement
- Servo motor control using PWM
- Object gripping and releasing
- Manual control through Arduino
- Real-time hardware interaction

## Technologies Used

- Arduino

- PWM

- Embedded Systems
- Electronics
